# Anna Bella

Site institucional desenvolvido durante o curso de Desenvolvimento Web
na Udemy, junto com o professor Jamilton Damasceno.

O projeto apresenta uma modelo fictícia chamada Anna Bella, contendo
páginas de:

-   Home\
-   Biografia\
-   Campanhas Publicitárias\
-   Contato

## Tecnologias utilizadas

-   HTML5\
-   CSS3

## Objetivo

Praticar estruturação de páginas com HTML e estilização com CSS,
aplicando os conceitos aprendidos em aula.
